Bien rechercher sur Internet

Fonctionnement d’un moteur de recherche

Un moteur de recherche explore les pages Web, les analyse et les classe selon leur pertinence.

Définitions essentielles

  • Internet : réseau mondial reliant des ordinateurs.
  • Web : service d’Internet permettant de consulter des pages.
  • Navigateur : logiciel pour afficher les pages Web.
  • URL : adresse d’une ressource sur Internet.

Astuces pour formuler une requête

  • Utiliser des mots-clés précis
  • Éviter les phrases longues
  • Utiliser les guillemets pour une expression exacte

Vérifier la fiabilité d’une information

  • Identifier la source
  • Comparer plusieurs sites
  • Vérifier la date
  • Identifier l’auteur
